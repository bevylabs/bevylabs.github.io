Our Steps to Change the World.

Feedback rewarding device for all kind of mobile application and games developers with satisfaction.

  • February 20, 2019
    • •
  • Ghost
    • •
  • BevyLabs

As the human, we used to learn mathematical operation since in childhood. This method provides a platform to learn mathematical operations in less amount of time.[1][2][3] This method contains two parts, a mobile application and the main device. Using voice recognition from the speaker ask questions from the user. Then the user provides answers and after checking the correctness. After checking the device give a reward like a candy dispenser. Finally, we will distribute the mobile application and this device, 3D printing files to the world and locally selected children. Then we can gather the data from the users. Using the above data we can prove that this method is efficient or not for the learning mathematical operations.

Objective: To analyse the effectiveness of the new device with a mobile application(special rewarding system) actually helps who are in an age group 5-10 years, to increase their mathematical knowledge. i) Maybe a rewarding device with that can connect with unity game engine through a library written by the researches. ii)Design a rewarding device using Computer Aided Design(CAD). iii)Design the mobile application that should with a rewarding device. iv)Distribute the 3D design and the mobile application to the world.
